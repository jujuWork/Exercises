HTML, CSS, and Javascript Technical Document

Introduction

HTML (HyperText Markup Language), CSS (Cascading Style Sheets), and JavaScript are the foundational technologies used to create and design web pages and web applications. HTML structures the content, CSS styles it, and JavaScript adds interactivity.


HTML (HyperText Markup Language)

Overview

HTML is the standard markup language for creating web pages. It describes the structure of a webpage using a series of elements, which are represented by tags.

Key Concepts

  • Elements and Tags: Elements are the building blocks of HTML pages, and they are defined by tags. For example, p defines a paragraph.

  • Attributes: Tags can have attributes that provide additional information about the element. For example, a href="https://example.com">Link uses the href attribute.

  • Document Obeject Model (DOM): HTML documents are structured as a tree of objects, which can be manipulated with JavaScript.

Common Elements

  • html : Root element of an HTML page.

  • head : Contains meta-information about the document.

  • title : Sets the title of the webpage.

  • body : Contains the content of the webpage.

  • h1 ~ h6 : Header tags.

  • p : Paragraph.

  • a : Anchor (link).

  • img : Image.

  • div : Division or section.

  • span : Inline container for text and other elements.


CSS (Cascading Style Sheets)

Overview

CSS is used to style and layout web pages. It controls the presentation, formatting, and layout of HTML elements.

Key Concepts

  • Selectors: Used to select the HTML elements to be styled.

  • Properties and Values: Define how selected elements should be styled. For example, color: red; sets the text color to red.

  • Cascading: Styles can cascade from parent to child elements, and multiple styles can be combined.

Common Selectors

  • Element Selector: Targets HTML elements by their tag name (e.g., p { ... }).

  • Class Selector: Targets elements by class attribute (e.g., .classname { ... }).

  • ID Selector Targets elements by id attribute (e.g., #idname { ... }).

  • Attribute Selector Targets elements by attribute (e.g., a[href] { ... }).


Javascript

Overview

JavaScript is a scripting language used to create and control dynamic website content. It allows for interactive elements, such as forms, animations, and user events.

Key Concepts

  • Variables: Containers for storing data values (e.g., var x = 5;).

  • Functions: Blocks of code designed to perform a particular task (e.g., function myFunction() { ... }).

Common Contructs

  • Variables: var, lets, const

  • Functions: function, arrow Functions () => {}

  • Conditional Statements: if else if else

  • Loops: for while do...while


Conclusion

HTML, CSS, and JavaScript work together to create dynamic and visually appealing web pages. HTML provides the structure, CSS handles the presentation, and JavaScript adds interactivity. Understanding how these technologies interrelate is essential for web development.
